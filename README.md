Overview:
Technorix-Test is a front-end project built with React and TypeScript.It demonstrates modern web development practices, including modular component structure, TypeScript type safety, and clean UI using Chakra UI.


Features:
Built using React (TypeScript) for a scalable and maintainable codebase

Chakra UI for fast, accessible, and responsive styling

Demonstrates best practices in component architecture and state management

Optimized for readability and performance


Tech Stack
Frontend: React, TypeScript

UI Library: Chakra UI
